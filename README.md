# 连锁超市智能网络系统

> 基于华为数通模拟设备ensp的连锁零售场景网络方案，实现“总部-仓储-门店”一体化管理与业务隔离。

---

## 📌 项目简介
为闪送连锁超市设计的一套**分层智能网络系统**，覆盖 1 个总部、1 个中心仓、4 个前置仓、12 家门店，解决原有网络混跑、高峰卡顿、难维护等问题，保障收银、仓储、办公等业务稳定运行。

---

## 🛠 核心技术栈

| 类别 | 技术 | 说明 |
|:----:|:----:|:----:|
| **架构** | 核心-汇聚-接入 | 分层解耦，提升转发效率 |
| **高可用** | VRRP + Eth-Trunk | 双机热备，链路冗余 |
| **路由** | OSPF | 动态选路，故障秒级切换 |
| **无线** | AC + Fit-AP | 统一管理，无缝漫游 |
| **安全** | VLAN + 防火墙 | 业务隔离，边界防护 |
| **运维** | eNSP + Wireshark | 仿真验证，流量分析 |

## 🖥 关键设备清单

| 层级 | 设备类型 | 型号 |

---

## ✅ 核心功能
- **五网隔离**：收银、办公、仓储、监控、访客物理/逻辑隔离。
- **无缝漫游**：仓库PDA、手持终端跨AP移动不掉线。
- **快速开局**：新门店采用配置模板，即插即用。
- **安全防护**：防火墙双机热备，防DDoS及越权访问。

## 🚀 快速部署（模拟器环境）
1. 打开华为 **eNSP**，拖入对应设备。
2. 导入 \`configs/\` 目录下的配置文件。
3. 启动设备，检查OSPF邻居与AP上线状态。
4. 使用STA终端连接Wi-Fi测试业务连通性。
